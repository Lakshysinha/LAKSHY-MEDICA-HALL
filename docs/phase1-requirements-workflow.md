# Phase 1 - Requirements & Workflow Mapping

## Roles
- Owner/Admin
- Pharmacist
- Staff/Billing

## Core Workflows
1. Add medicine with full metadata, batch, MFG/EXP validation.
2. Search medicine by name/batch/code (scanner-equivalent code input).
3. Auto short-list when quantity <= 3.
4. Sales entry with strips/tablets, payment mode, optional customer name.
5. Daily dashboard with total sold units and payment splits.
